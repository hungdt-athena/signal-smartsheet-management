Weekly Feedback — evaluator differentiation

Same data, three ways to make different evaluators read as distinct blocks. Pick one (or mix). Colors are derived per evaluator name.

Option AAccent edge + avatar

A colored left edge runs down each evaluator's rows, with a square initials avatar. Quiet; the table stays white. Best if you want subtle separation without heavy color.
